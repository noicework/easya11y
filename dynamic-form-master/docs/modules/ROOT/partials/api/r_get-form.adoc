= Get a specific form

[.d-get]#GET# `/forms/:id`

Description::
Returns details of a given form. Returns the whole form structure.

== Request

[source,sh]
----
GET {{host}}{{base_path}}/forms/1
Authorization: Basic {{username}} {{password}}
Content-Type: application/json
----
