= Get form results

[.d-get]#GET# `/forms/:id/results`

Description::
Returns a list of all form responses (results).

== Request

[source,sh]
----
GET {{host}}{{base_path}}/forms/1/results
Authorization: Basic {{username}} {{password}}
Content-Type: application/json
----
