= Get all forms

[.d-get]#GET# `/forms`

Description::
Returns a list of all forms defined in the system.

== Request

[source]
----
GET {{host}}{{base_path}}/forms/
Authorization: Basic {{username}} {{password}}
Content-Type: application/json
----
